Install path

Build, launch, install

Dashboard served by the native host Diagnostics and setup mode Hosted weather-only preview 
1. Open PowerShell in app
2. Run: powershell -File publish.ps1
3. Launch: ..\publish\XenonEdgeHost.exe
4. Optional auto-start: powershell -File install.ps1
To create the easy installer:
Double-click: Build XENEON Installer.cmd

Output:
app\dist\XenonEdgeHost-Setup-<version>-<date>.exe

Legacy bridge files are isolated in the bridge folder; the native host is the product path.
