# Support

Customer support starts in the app at `support.html`.

Current free beta support path:

- GitHub Issues: https://github.com/SilverFuel/xeneon-widgets/issues/new/choose
- Downloads and release notes: https://github.com/SilverFuel/xeneon-widgets/releases
- Free beta license policy: `refund-policy.html`

Before a paid public launch, add monitored `support@...` and `security@...` inboxes to the support page and checkout emails. Do not invent inboxes that are not actually monitored.

When helping a customer, ask for:

- app version
- Windows or macOS version
- whether the dashboard opens at `http://127.0.0.1:8976/`
- a screenshot of Diagnostics
- whether Weather, Hue, UniFi, or launchers were configured

Windows logs and settings are usually under:

```text
%LOCALAPPDATA%\XenonEdgeHost
%APPDATA%\XenonEdgeHost
```

Customers can remove local dashboard settings and protected integration secrets from the in-app Privacy or Setup panel with Reset all app data. They can also use Start Menu > XENEON Edge Host > Uninstall and Remove Local Data when removing the app.

Do not ask customers to send API keys, Hue tokens, calendar feed secrets, or other private credentials.
