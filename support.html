Install and Open

Install the app, open it from the Start Menu, and finish the setup screen. The dashboard should load at http://127.0.0.1:8976/.

Normal path

Reset App Data

Open the Privacy or Setup panel, choose Reset all app data, then confirm. When uninstalling, use the Start Menu cleanup shortcut to remove local app data too.

Security

Do not send API keys, Hue tokens, calendar feed secrets, or screenshots that reveal private device data. Report private security issues through GitHub Security Advisories.

Report Security Issue
The free beta uses GitHub Issues and GitHub Security Advisories as the live support paths. A paid launch should add monitored support and security inboxes before taking customer money.
